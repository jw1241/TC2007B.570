# Backend - Equipo 4
Este es el repositorio base para el desarrollo del backend del proyecto con el socio-formador. Actualmente se encuentra en su fase inicial de estructura y configuracion de servicios basicos.

## Estado del Proyecto
Esta entrega comprende la configuracion inicial del entorno, la arquitectura de carpetas y la definicion de las rutas principales para la API.

## Tecnologias
* **Node.js**: Entorno de ejecucion para JavaScript orientado a servidores.
* **Express.js**: Framework para la creacion de servicios web y APIs.
* **Swagger**: Herramienta para la documentacion interactiva de los endpoints.
* **NPM**: Manejador de paquetes para la gestion de dependencias.
